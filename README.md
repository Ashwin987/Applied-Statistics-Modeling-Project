## Overview
This project analyzes how student lifestyle factors influence academic performance. The goal is to predict studentsâ€™ GPA based on variables related to sleep, extracurricular activities, social time, and study hours using linear regression models.

## Data
The dataset includes student-level information on:
- Sleep hours
- Study hours
- Social hours
- Extracurricular activity involvement
- GPA (response variable)

## Methods
- Exploratory Data Analysis (EDA)
- Multiple Linear Regression
- Model interpretation and diagnostics

## Key Findings
- Study time showed the strongest positive association with GPA.
- Excessive social time was negatively associated with academic performance.
- Sleep and extracurricular activities contributed modest but meaningful effects.
